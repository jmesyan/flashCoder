React.js server-side rendering with Go
=====

This experiment is based on [otto](https://github.com/robertkrimen/otto), a Javascript interpreter for Go.

To run:
```bash
go build ./...
./go-reactjs
```

To benchmark:
```bash
go test -bench=.
```
